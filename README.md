# Python_based_image2cartoon
Image & Webcam Cartoonizer using Python
 Project Overview

This project is a Python-based Image and Webcam Cartoonizer that converts real images or live webcam frames into a cartoon-style output using image processing techniques. It demonstrates practical use of OpenCV for real-time image capture and transformation.

Features:-
Image cartoonization
Real-time webcam cartoon filter
Input image processing
Save cartoonized output image
Fast and lightweight processing

Technologies Used:-
Python
OpenCV (cv2)
NumPy
Webcam (for live capture)
